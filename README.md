# SIPI 4 - Sistem Peminjaman Tempat

## Deskripsi
Sistem Peminjaman Tempat (SIPI 4) adalah aplikasi berbasis web yang dirancang untuk mengelola peminjaman gedung dan penjadwalan peminjaman ruangan di Politeknik Negeri Cilacap secara lebih efisien.

Latar belakang pengembangan aplikasi ini adalah proses pengajuan peminjaman yang masih dilakukan secara manual, termasuk pencatatan jadwal yang menggunakan papan tulis. Hal tersebut sering menyebabkan data peminjaman terhapus, bentrok jadwal, serta terjadinya miskomunikasi antar pengguna.

Melalui SIPI 4, proses peminjaman menjadi lebih terstruktur, terdokumentasi, dan meminimalkan terjadinya konflik penggunaan ruangan.

## Tim
Project ini dikerjakan oleh 3 orang dalam pembagian fitur masing-masing. Berikut adalah nama anggota kelompok beserta dengan pembagiannya:
1. Salwa Roudhotul Ilmi (Login dan User)
2. Rosta Iranti Putri (Home dan Gedung)
3. Aisyah Nurul Laila (Jadwal dan Peminjaman)
